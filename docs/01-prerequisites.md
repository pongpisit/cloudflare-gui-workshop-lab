# Module 1: Prerequisites

Set up everything you need before starting the workshop.

**Time needed: 10 minutes**

---

## What You Will Do

1. Create a free Cloudflare account
2. Verify access to Cloudflare Dashboard
3. Verify access to Zero Trust Dashboard

---

## Step 1: Create a Cloudflare Account

### 1.1 Open the Sign-Up Page

**Copy this link and paste it in your browser:**
```
https://dash.cloudflare.com/sign-up
```

### 1.2 Fill in Your Details

1. Enter your **email address**
2. Create a **password**
3. Click the **Create Account** button

### 1.3 Verify Your Email

1. Open your email inbox
2. Find the email from Cloudflare
3. Click the **verification link** in the email

> **Done!** You now have a Cloudflare account.

---

## Step 2: Verify Dashboard Access

### 2.1 Access Cloudflare Dashboard

**Copy this link and paste it in your browser:**
```
https://dash.cloudflare.com/
```

You should see the Cloudflare Dashboard. Look for **"Workers & Pages"** in the left menu.

#### ✅ Verification:

- [ ] You can log in to the Dashboard
- [ ] You can see "Workers & Pages" in the left sidebar

---

## Step 3: Verify Zero Trust Dashboard Access

### 3.1 Access Zero Trust Dashboard

**Copy this link and paste it in your browser:**
```
https://one.dash.cloudflare.com/
```

### 3.2 First-Time Setup (if prompted)

If this is your first time, the system will prompt you to create a Zero Trust organization:

1. Set team name: e.g., `my-workshop-org`
2. Select plan: **Free**
3. Fill in the information as requested

#### ✅ Verification:

- [ ] You can access the Zero Trust Dashboard
- [ ] You can see the left sidebar menu: Networks, Traffic policies, Insights, etc.

---

## What You Need

| Item | Description | Required |
|------|-------------|----------|
| **Cloudflare Account** | Created in Step 1 | ✅ Required |
| **Web Browser** | Chrome, Edge, or Firefox (latest version) | ✅ Required |
| **Internet Connection** | Stable connection, able to access `*.cloudflare.com` | ✅ Required |
| **GitHub Account** | For deploying Pages via GitHub integration | ⭐ Recommended |
